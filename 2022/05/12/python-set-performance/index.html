set创建方式

两种创建实例方式反汇编

可以看出，通过字面值去创建一个集合，在解释器实现上是通过调用了一个叫做 BUILD_SET 的方法直接创建；

而使用 set() 方法创建集合，解释器实现上会先通过 set 的名字去寻找构造方法，并且调用 BUILD_LIST 方法去创建一个 list 传入 set 构造方法，从而创建一个集合。

所以结论为：使用字面值直接创建集合要比使用 set 构造方法效率更高。
