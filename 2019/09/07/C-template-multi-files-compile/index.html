按照常理，正常使用g++编译即可生成main可执行文件，但是在执行最后一步链接的时候会出现问题。

可以看到，单独编译实现代码的时候是可以通过的，但是在两个文件链接起来的时候出现了问题。经过查找资料，貌似是因为实现文件中实现的是cycleQueue<T>这个类型，但是main中使用了cycleQueue<int>这个类型，所以编译器找不到这个类型的定义。
